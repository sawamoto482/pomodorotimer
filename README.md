# ポモドーロタイマー

シンプルなポモドーロタイマーのWebアプリケーションです。

## 機能

- 作業モード（25分）
- 短い休憩モード（5分）
- 長い休憩モード（15分）
- スタート、一時停止、リセット機能
- タイマー終了時の音声通知

## 使用方法

1. ブラウザでアプリケーションを開く
2. 作業モード、短い休憩、長い休憩のいずれかを選択
3. 「開始」ボタンをクリックしてタイマーを開始
4. 「一時停止」ボタンでタイマーを一時停止
5. 「リセット」ボタンでタイマーをリセット
